## Project Summary

In this capstone project, your team is responsible for:

1. Processing DIA proteomics data (`.mzML` files)
2. Performing protein/precursor identification using the assigned software tool
3. Generating a valid SDRF file describing the experimental design
4. Submitting identification results and metadata according to course guidelines
5. Performing downstream statistical analysis

⚠️ The preferred output level for identification is **precursor/PSM-level data**, in addition to protein-level summaries.
